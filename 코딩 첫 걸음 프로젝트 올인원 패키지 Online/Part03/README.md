## CSS란?

- CSS는 Cascading Style Sheets의 약자이다.
- CSS는 HTML 요소가 화면, 종이 또는 기타 미디어에 표시되는 방식을 설명한다.
- CSS는 많은 작업을 절약한다. 한 번에 여러 웹 페이지의 레이아웃을 제어할 수 있다.
- 외부 스타일시트는 CSS 파일에 저장된다.
